# AeroVault

AeroVault is an intelligent, self-hosted file management platform that unifies your cloud storage and enhances your workflow with AI-powered insights.

## Features

*   **Intelligent Organization:** AI-powered tagging and categorization automatically organize your files, making them instantly searchable.
*   **Unified Search:** Find any file, across all your linked cloud services, with one simple, powerful search.
*   **Unyielding Security:** Keep control of your data with a self-hosted solution, complete with access controls and detailed activity logs.

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/AeroVault.git
    ```
2.  Install the dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the migrations:
    ```bash
    python manage.py migrate
    ```
4.  Run the development server:
    ```bash
    python manage.py runserver
    ```

## Running the tests

To run the tests, use the following command:

```bash
python manage.py test
```
